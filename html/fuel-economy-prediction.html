Fuel efficiency is a critical factor in vehicle design, cost optimization, and environmental sustainability. This project converts multi-year fuel economy data into actionable insights using data science and machine learning.

  • How has vehicle fuel efficiency evolved over time?
  • Which vehicle characteristics most influence City MPG?
  • Can ML models predict and classify eco-friendly vehicles?

Why This Project?

Rather than relying on assumptions, this project analyzes real-world fuel economy datasets across multiple years to understand trends and performance using machine learning.

Objectives

  • Analyze fuel economy trends over 10 years
  • Study eco-friendly (SmartWay) vehicles
  • Predict City MPG using regression models
  • Classify vehicles as SmartWay / Non-SmartWay
  • Compare model interpretability

Dataset Description

  • City MPG
  • Highway MPG
  • Combined MPG
  • Cylinders
  • Fuel Type
  • Vehicle Class
  • SmartWay Indicator

Methodology

Step 1: Data Cleaning

Standardized columns, handled missing values, and validated data.

Step 2: Dataset Integration

Merged yearly datasets and added a year column.

Step 3: ML Preprocessing

Feature encoding, scaling, and train-test splitting.

Step 4: Linear Regression

Predicted City MPG using mileage-related features.

Step 5: Classification Models

Logistic Regression and Decision Tree for SmartWay classification.

Key Results

  • Fuel efficiency improved consistently over the years
  • SmartWay vehicle adoption increased
  • Linear Regression showed strong predictive ability
  • Classification models were interpretable and stable

Key Learnings

  • Data quality matters more than model complexity
  • Simple models can be highly effective
  • Interpretability is as important as accuracy

Conclusion

This project demonstrates how structured preprocessing and interpretable machine learning models can transform raw automotive data into meaningful insights.
