This project focuses on predicting medical insurance charges using supervised machine learning. Healthcare expenses vary significantly based on demographic and lifestyle factors, making cost estimation a meaningful real-world regression problem.

Using the Kaggle Insurance Dataset, a Linear Regression model was trained to estimate individual medical charges based on structured tabular health data.

  • Can insurance charges be predicted using a linear model?
  • Which variables most strongly influence cost?
  • How much variance in charges can be explained by basic features?

Why This Project?

Insurance pricing is influenced by both behavioral and physiological variables. Understanding how these variables impact cost provides insights into risk modeling and healthcare analytics.

This project was built to explore:

  • Linear relationships between health attributes and cost
  • Impact of smoking behavior on medical charges
  • Performance of a baseline regression model on tabular data

Objectives

  • Prepare and clean structured healthcare dataset
  • Encode categorical variables appropriately
  • Split data into training and testing sets (80/20)
  • Train Linear Regression model
  • Evaluate performance using R¬≤ score
  • Interpret model behavior and feature influence

Dataset Description

Source: Kaggle Insurance Dataset

  • Age
  • Sex
  • BMI
  • Children
  • Smoker (Yes/No)
  • Region

Target Variable: Charges (Medical Insurance Cost)

Methodology

Step 1: Data Preprocessing

Loaded dataset using Pandas. Categorical variables were encoded and structured into numerical format suitable for regression.

Step 2: Train-Test Split

Dataset was split into 80% training and 20% testing using scikit-learn's train_test_split.

Step 3: Model Training

Linear Regression model was trained using scikit-learn to estimate medical insurance charges.

Step 4: Evaluation

Model performance was evaluated using R¬≤ score to measure explained variance.

Key Results

  • R¬≤ Score: 0.743
  • Model explains approximately 74% of variance in insurance charges
  • Smoking status had significant impact on predicted cost
  • Linear Regression provided strong baseline performance

Key Learnings

  • Structured tabular data can be effectively modeled with simple regression
  • Behavioral variables (like smoking) dominate cost prediction
  • Interpretability is valuable in healthcare-related ML problems
  • Baseline models are important before moving to complex algorithms

Conclusion

This project demonstrates how a well-prepared dataset and a baseline Linear Regression model can produce meaningful cost predictions in a healthcare insurance setting. It highlights the importance of feature understanding, preprocessing, and interpretability in applied machine learning.
